# Contributing to KITT Gateway

Thanks for your interest. This project is in early development and actively evolving.

## Current Status

KITT Gateway is a solo project in active development. The architecture is stabilizing but APIs and interfaces may change without notice.

## How to Participate

**Issues** — Bug reports, feature requests, and architecture discussions are welcome. Please open a GitHub issue.

**Pull Requests** — Not currently accepted without prior discussion. If you'd like to contribute code, open an issue first describing the change and we'll discuss approach.

**Security Issues** — If you find a security vulnerability, please do **not** open a public issue. Email the maintainer directly (see GitHub profile) so it can be addressed before disclosure.

## Development Setup

See the [README](README.md) for setup instructions. The project runs on Linux with Docker, an NVIDIA GPU, and Python 3.12+.
